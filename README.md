# The Love Calculator

The Love Calculator is a simple and fun API that can be used to build a free romantic partner compatibility application. Matchmaking services are provided by this API using analysis of the names of the couple involved. The output tracks the probability of success for a romantic relationship.

### Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Json
- Vercel
- Love Calculator API
